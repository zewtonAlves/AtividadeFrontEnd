História da Internet

Como tudo começou

A Internet surgiu a partir da ARPANET, criada nos anos 1960 durante a Guerra Fria, para conectar centros de pesquisa e universidades nos Estados Unidos.

A primeira conexão foi estabelecida em 1969 entre a UCLA e o Stanford Research Institute.

"A ARPANET foi a base para o que viria a se tornar a Internet moderna."

Como a Internet Funciona

A Internet é uma rede mundial de computadores interligados. Quando você acessa um site, seu dispositivo envia um pedido que percorre diferentes etapas até chegar ao servidor.

Esse caminho pode ser descrito assim:

  1. Servidor
  2. Provedor de Internet
  3. Roteador
  4. Dispositivo do Usuário

Termos Importantes

IP
Identificação única de cada dispositivo na rede.
DNS
Sistema que traduz nomes de domínios em endereços IP.
HTTP
Protocolo de transferência de hipertexto.
HTTPS
Versão segura do HTTP, com criptografia.
